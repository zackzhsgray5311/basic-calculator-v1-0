## Overview

Basic Calculator is a simple calculator that runs in the browser and focuses on basic arithmetic tasks. The interface is powered by HTML, CSS, and JavaScript, with DOM updates and event listeners driving button interactions in a clean, direct layout.

It serves well as a compact frontend example for learning calculator behavior and simple web interactivity. You can use it as a study aid, a classroom reference, or a base for adding extra calculator capabilities to a lightweight web app.

---

## What it includes

- Performs addition, subtraction, multiplication, and division
- Provides an AC control to clear the current calculation
- Provides a DEL control to delete the most recent character
- Uses HTML for page structure, CSS for presentation, and JavaScript for interactivity
- Relies on DOM manipulation to refresh the display in real time
- Uses event listeners to process calculator button input
- Keeps the calculator logic entirely in the frontend
- Works in the browser without additional runtime dependencies

## How to use it

1. Open the calculator in a browser.
2. Input values with the on-screen buttons.
3. Pick an operator such as +, -, x, or /.
4. Press = to display the answer.
5. Use AC to clear the session.
6. Use DEL to erase the most recent entry when fixing an input.

Example workflow:

- Click 12
- Select +
- Click 8
- Press =
- Result appears in the display
